Hint - Dealing With Unstructured Code

The program from this problem is an example of unstructured code, where loops and blocks of conditional code are implemented with goto statements, which perform an unconditional jump to another place in the program. When reverse-engineering a program like this, it helps to know how high-level constructs look when implemented with goto. Most constructs can be represented in several ways, but these are some of the more common formats.

While loop:
begin:
if(!condition) goto end;
loop body;
goto begin;
end:
Rest of program...
If-then-else statement:
if(condition) goto then;
else-statements;
goto end;
then:
then-statements;
end:
Rest of program...
